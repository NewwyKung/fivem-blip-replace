# Fivem Blip Replace Texture

This script allows you to replace blip textures in the game.

## Installation

1. Add the script to your resources folder.
2. Add `ensure fivem-blip-replace` to your server.cfg.

## Configuration

You can configure the script in the `config.lua` file. and Replace the blip textures in the `assets` folder.

## Usage

The script will automatically replace the blip textures when the resource starts or ensure.
